# Proyecto HireSphere Recursos Humanos

## Descripción
El proyecto HireSphere Recursos Humanos es una aplicación web que permite a los usuarios buscar empleo y a las empresas publicar ofertas de trabajo. La aplicación cuenta con un sistema de autenticación de usuarios,
 un buscador de empleo, un listado de ofertas de trabajo y un formulario para publicar ofertas de trabajo.

 Temas completos en esta entrega:
 * Administración Novedades
    * Ofertas de trabajo
    * Registro de usuarios
    * Inicio de sesión
    * Publicación de ofertas de trabajo (solo para usuarios autenticados como reclutadores)

    Dependiendo del rol con el que se loguea el usuario muestra un menú de perfil diferente.

    El administrador puede publicar noticias
    El reclutador puede publicar ofertas de trabajo
    El postulante puede postularse a las ofertas de trabajo
